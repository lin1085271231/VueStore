## 自学前端路上的第一个小项目

 - 下载之后先安装依赖包
 - 终端下运行 **`npm install`**
 - 安装完成后运行 **`npm run dev`** 即可
 ## 所使用的
 - webpack
 - vue-router
 - vuex
 - axios
 - Mint-UI
 - MUI

 ## 功能
 
 - 新闻资讯
 - 图片分享
 - 商品购买
 - 购物车功能
